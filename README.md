# Smart Health Solutions

An AI-powered health recommendation system that provides personalized diet plans and yoga recommendations based on user health profiles and chronic diseases.

## 🌐 Live Demo

https://smarthealth-solutions-1.onrender.com

## 🚀 Features

- Personalized diet recommendations
- Yoga pose suggestions
- BMI calculation
- Chronic disease-based wellness plans
- User profile management

## 🛠️ Tech Stack

- Python
- Flask
- Scikit-learn
- SQLite
- HTML/CSS
- Bootstrap
